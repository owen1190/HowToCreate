# HowToCreate系列

## 前言

本人在读小硕一枚，正在学习前端知识，之前找实习时，发现各大公司招前端工程师，除了考察基本计算机知识和编程题，也会让应聘者实现一些小的demo，所以特此建此库让各位应聘者能提前预习一下，也能让正在入门前端的学习者，通过HowToCreate系列更加熟悉JavaScript语法。
